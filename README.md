## 【功能介绍】
**本系统包含管理员、教师、用户几个角色。
管理员：学生管理、教师管理、学科信息管理、课程管理、课程视频管理、课程资料管理、试卷管理、试题管理、系统管理、考试管理等功能。
校友：科信息管理、课程管理、课程视频管理、课程资料管理、试卷管理、试题管理、系统管理、考试管理等。
用户：查看课程、查看资料、查看试卷列表、个人中心。**
